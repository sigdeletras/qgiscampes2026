📡 Fase 1: Script Independiente (Fuera de QGIS) ▼
Paso 1.1: Descargar y mostrar municipios ▼
Escribe un script de Python INDEPENDIENTE (que NO use QGIS) que acceda al ATOM de parcelas (https://www.catastro.hacienda.gob.es/INSPIRE/CadastralParcels/ES.SDGC.CP.atom.xml), extraiga todos los municipios y los muestre en consola con formato "CÓDIGO - NOMBRE". Usa requests y xml.etree.ElementTree. Incluye comentarios.

Ejecutar:

python test_parcelas.py
Paso 1.2: Buscar municipio y obtener URL ▼
Extiende el script. Añade función que pida al usuario escribir un municipio, lo busque (sin distinguir mayúsculas), muestre coincidencias, extraiga la URL del ZIP y la imprima.
Paso 1.3: Descargar y descomprimir ▼
Añade descarga del ZIP, guardar en carpeta "descargas/", descomprimir, mostrar archivos extraídos, identificar el archivo .gml.
Paso 1.4: Validar GML ▼
Añade función que lea el GML, muestre primeras 10 líneas, cuente etiquetas, verifique que no está vacío y guarde resumen. Dame el script COMPLETO final.
🧩 Fase 2: Plugin Básico en QGIS ▼
Paso 2.1: Plugin "Hola Mundo" ▼
Dame código completo para plugin QGIS "catastro_loader" con menú "Catastro", item "Cargar parcelas" que muestre un QMessageBox con "Plugin funcionando". Incluye __init__.py, catastro_loader.py, metadata.txt, resources.py.
Paso 2.2: Integrar lógica de descarga ▼
Integra el script test_parcelas.py en el plugin. Usa QInputDialog para municipio, QProgressDialog para progreso, QMessageBox para resultado. Ejecuta en QThread para no congelar QGIS.
Paso 2.3: Cargar GML en QGIS ▼
Añade función que localice el .gml, cargue como QgsVectorLayer, añada al proyecto, haga zoom, maneje errores y opcionalmente elimine ZIP temporal.
🖥️ Fase 3: Interfaz de Usuario Completa ▼
Paso 3.1: Diálogo con Qt ▼
Crea un QDialog con: QComboBox (Parcelas/Direcciones/Edificios), QLineEdit (municipio), botón Examinar para carpeta, QCheckBox "Eliminar ZIP", botones Descargar/Cancelar, QProgressBar. Guarda carpeta con QSettings.
Paso 3.2: Conectar los 3 ATOMs ▼
Modifica para usar ATOM según selección: Parcelas (CP), Direcciones (AD), Edificios (BU). Refactoriza lógica común en función descargar_datos(tipo, municipio, carpeta).
Paso 3.3: Autocompletado ▼
Añade QCompleter al campo municipio. Al escribir 3 caracteres, buscar del ATOM actual. Cachear lista para no recargar.
🛡️ Fase 4: Robustez y Calidad ▼

Paso 4.2: Refactorizar ▼

🔧 Anexo: Solución Rápida para Errores Frecuentes

Error: "No module named requests" ▼ 
pip install requests
C:\ruta\QGIS\bin\python.exe -m pip install requests
Error: "No provider for layer" al cargar GML ▼
Dame un código de ejemplo que convierta un GML a GeoPackage usando PyQGIS. Así puedo cargar el GeoPackage en lugar del GML directamente.
Error: Plugin no aparece en QGIS ▼
  • Verifica que la carpeta se llama exactamente igual que el plugin
  • Verifica metadata.txt tiene formato correcto
  • Reinicia QGIS completamente
  • Plugins → Administrar → Configuración → Ver logs
